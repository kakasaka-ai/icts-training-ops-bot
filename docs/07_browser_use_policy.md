# Browser-use policy

## Basic policy

Use browser-use only when a stable API or CSV export is unavailable or not yet approved.

Start with read-only tasks.

## Allowed read-only use cases

- ICTS AI registration status check.
- LINE admin screen tag count check.
- RaySheet screen check when Salesforce API access is unavailable.
- Money Forward status check when API or export is unavailable.

## Prohibited without explicit approval

- Submitting production forms.
- Sending LINE messages.
- Updating Salesforce or RaySheet through the browser.
- Deleting ICTS AI accounts.
- Removing Slack users or deleting workspaces.
- Changing Google Drive permissions.
- Creating Money Forward payment requests.

## Credential rules

Do not store passwords, 2FA codes, browser sessions, cookies, or screenshots containing sensitive data in this repository.
